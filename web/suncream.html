SUNCREAM

선크림이란?

태양의 자외선(UV)으로부터 피부를 보호하기 위해
인공적으로 만들어진 원료를 물리적 화학적 작용을 통하여 만든 화장품

선크림을 사용하기 귀찮아서

"피부 좀 타고 말지"

라는 생각을 하실 것 입니다.
하지만 선크림의 이점은 이것 뿐만이 아닙니다.

그럼 선크림을 사용해서 얻는 이점은 무엇일까요?
피부 톤 유지
노화 방지(주름)
색소침착 방지
피부암 예방
기초 메이크업

TYPE

선크림의 종류

01
무기자차
무기물질을 이용해 자외선을 튕겨내는 방법
자극이 적으며 바르는 즉시 자와선 차단 효과 생김
발림성이 뻑뻑하며 백탁 현상 발생
02
유기자차
화학적 성분을 이용해 자외선을 흡수하여
다시 열로 배출하는 방식
발림성이 좋고 백탁 현상이 없음
눈시름과 피부에 자극이 있을 수 있음
03
혼합자차
무기자차와 유기자차를 혼합한 선크림
백탁이 적으며 부드러운 발림성
다소 높은 가격대로 형성
04
선스틱
스틱 방식으로 피부에 덧바르는 형식
손에 묻지 않고 간편하게 사용 가능
청결을 유지하기 힘듦
